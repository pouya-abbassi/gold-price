# gold-price

This application helps you get current gold price (in Rials/10) from these gold dealers:
  - talanews.com
  - iranshemsh.com
  - parsisgold.com

Using:
  - [python] 3.6.4
  - [kivy] 1.10.1
  - [urllib] 21.5

### How to run?
```sh
python main.py
```
